What this site contains

The source notes are maintained in main.tex. GitHub Pages publishes a small site whose landing page lives at the root and whose generated blueprint is served under /blueprint/.

The graph view is generated from labels and \uses{...} annotations in the LaTeX source, so the site doubles as both lecture notes and a structural map of the material.
